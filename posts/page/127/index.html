rPath Appliance Agent 1.0.9 in the wild!
Jan 01, 0001
Today we have released rPath Appliance Agent (rAA) version 1.0.9!!! I’m very excited about this since it has 2 new features that I worked on: A new plugin for setting the time, date and timezone, and configuring the Network Time Protocol (NTP). The rAA user password is now salted to provide additional security. A long time ago I experienced the joy of seeing a program I had developed and open sourced pick up over 30000 downloads on downloads.
